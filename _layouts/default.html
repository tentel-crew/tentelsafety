TentelSafety Documentation

Welcome!

TentelSafety provides a security feature for the Discord server. The bot is built on algorithms and functions that initiate an addition to the secure environment on the server. The algorithms include: - Anti DDoS, saving the server and minimizing the threat from all known Nuke bots. - Negative actions against alternative accounts. - Monitoring the activity status of the incoming user on the server. - Control of the rights of the incoming user to the server. - Quarantine that eliminates potential threats from users in the database. The functions include: - User verification using heuristic analysis. - Tentel Key, an additional configuration that sets the access key for the algorithm configurator. - Quick Ban, the command responsible for banning several users at the same time by IDs. And so on... Bypass TentelSafety. Currently, there are no malicious integrations capable of bypassing the Anti DDoS TentelSafety. But there is a technical tool that is able to bypass the identification of the attack; the bot does not respond to self bots in the attacker's accounts. To counteract this, it is enough for you not to grant administrator rights, server management, webhook management, role management and channel management to questionable persons. (details of information security in Discord can be found on the Tentel Crew server)
